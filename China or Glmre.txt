#include <iostream>
using namespace std;
int main()
{
	char c1 = 'C', c2 = 'h', c3 = 'i', c4 = 'n', c5 = 'a';
	c1 = c1 + 4, c2 = c2 + 4, c3 = c3 + 4, c4 = c4 + 4, c5 = c5 + 4;
	cout << c1 << c2 << c3 << c4 << c5 << endl;
	return 0;
}
In order to transform 'China' into a secret code, you need to replace the original letter
with the new letter which is the fourth one after the old letter.So 'China' can be 
referred as 'Glmre'.It is very easy.
